Option A

Spine

Vertical spine, each post a node on the machine. State ids. Editorial display title, body description. Dot fills accent on hover. Calmly on-brand — reads as a side-view of the writing state's substates.

Option B

Ledger

Monospace log output. Aligned columns: ISO date · state id · title. Reads like a process journal or kernel log — dev-native, dense, very distinctive. The pipe gutter keeps it feeling like a continuous stream rather than discrete cards.

Option C

Timeline — zoomable

Posts positioned on a 2D canvas by date. Titles dodge above and below the chronology to avoid overlap; pan and zoom when clusters get dense. Built on React Flow — same node/edge model as the nav machine, just scaled to the archive. The MiniMap in the top right gives you overview while you're zoomed in; the controls in the bottom right reset the view.
